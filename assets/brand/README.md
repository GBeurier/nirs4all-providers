# nirs4all-providers brand assets

Logo assets for `nirs4all-providers`, following the ecosystem brand system whose
single source of truth is **nirs4all-org** (`assets/_chart/Logos.txt` +
`assets/brand/`). The square icon is the same spectral-wave mark every package
shares, recolored to this package's key.

## Files

| File | What it is |
|---|---|
| `icon.svg` | Square app/brand mark — spectral wave, white content, on the package color. Self-contained, font-free. |
| `stacked.svg` | Icon over the `n4v` lettermark (light background). |
| `stacked-dark.svg` | Same, on-dark variant (`n` → white). |
| `horizontal.svg` / `horizontal-dark.svg` | Icon plus `nirs4all-providers` outlined wordmark. |
| `*.png`, `favicon.ico`, `og.png` | Raster kit generated from the same source. |

The full kit is generated by the nirs4all-org pipeline
(`assets/_chart/build/gen.py`, `cairosvg` + `fonttools` + the Inter font) and
mirrored into this repo. It is intentionally not hand-authored here.

## Package key

`nirs4all-providers` is registered in the nirs4all-org brand table
(`assets/_chart/Logos.txt`) with these tokens:

| field | value | rationale |
|---|---|---|
| color | `#D946EF` (RGB `217, 70, 239`) | Distinct from the existing data/repository/papers colors and visually readable as a connector/read-layer surface. |
| lettermark | `n4v` | The spec assigns `n4<first-letter>`, but providers' initial `p` is already taken by `nirs4all-papers` (`n4p`). `v` is the first free in-word letter (pro**v**iders), so the mark is unambiguous. |

Constant accents follow the ecosystem rule: `n` is black `#000000` (white on
dark), the `4` is always red `#E9362D`, the third glyph is the package color.

SVG wordmarks are outlined, so the kit has no runtime font dependency.
